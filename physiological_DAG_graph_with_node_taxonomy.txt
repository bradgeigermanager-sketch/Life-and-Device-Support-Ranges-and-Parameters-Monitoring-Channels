Physiological DAG graph (high‑level overview)

| Layer            | Role                                      | Examples                                      |
|------------------|-------------------------------------------|-----------------------------------------------|
| Sensors          | Raw acquisition                           | HR, SpO₂, RR, Core Temp, IMU, EEG, EMG        |
| Derived metrics  | Feature extraction / fusion               | HRV, Shock Index, Exertion Index, Fatigue     |
| Composite states | Health/condition classification           | Oxygenation State, Thermal Stress, Hydration  |
| Control loops    | Actuation & policy                        | O₂ control, HVAC, workload, hydration alerts  |
| System outputs   | Commands, alerts, logs                    | Setpoints, alarms, trend records              |

---

Node taxonomy

Sensor nodes (raw)

- HR: phys.vitals.hr
- SpO₂: phys.vitals.spo2
- Respiration Rate: phys.resp.rr
- Core Temperature: phys.temp.core
- Skin Temperature: phys.temp.skin
- Blood Pressure: phys.hemo.bp
- ECG Waveform: phys.vitals.ecg
- EEG Bands: phys.neuro.eeg_bands
- IMU Motion: phys.motion.imu
- EMG: phys.motion.emg
- End‑Tidal CO₂: phys.resp.etco2
- Hydration Index: phys.hydration.index
- Electrolytes (Na/K): phys.hydration.na, phys.hydration.k
- Sweat Rate: phys.hydration.sweat_rate
- Blood Glucose: phys.metabolic.glucose
- Lactate: phys.metabolic.lactate
- Hemoglobin: phys.metabolic.hb

Derived metric nodes

- HRV: phys.derived.hrv
- Shock Index: phys.derived.shock_index
- Exertion Index: phys.derived.exertion_index
- Gait Stability: phys.derived.gait_stability
- Thermoregulatory Stress: phys.derived.thermal_stress
- Oxygenation Reserve: phys.derived.oxygenation_reserve
- Cognitive Load: phys.derived.cognitive_load
- Fatigue Index: phys.derived.fatigue_index
- Metabolic Rate: phys.derived.metabolic_rate
- Hydration Status: phys.derived.hydration_status

Composite state nodes

- Cardiovascular Status: state.cv
- Respiratory Status: state.resp
- Thermal Status: state.thermal
- Hydration Status (macro): state.hydration
- Fatigue / Workload Status: state.fatigue
- Overall Health State: state.overall_health

Control loop nodes (actuation targets)

- O₂ Control: ctrl.env.o2
- CO₂ Scrubber Control: ctrl.env.co2_scrubber
- HVAC / Thermal Control: ctrl.env.thermal
- Workload Scheduling: ctrl.ops.workload
- Hydration Policy: ctrl.ops.hydration
- Alerting / Alarms: ctrl.ops.alerts

---

Edge set (dependencies)

I’ll express the DAG as an adjacency list: source -> [targets...].

Sensor → derived metrics

- phys.vitals.hr →  
  - phys.derived.hrv  
  - phys.derived.shock_index  
  - phys.derived.exertion_index  

- phys.vitals.ecg →  
  - phys.derived.hrv  

- phys.vitals.spo2 →  
  - phys.derived.oxygenation_reserve  

- phys.resp.rr →  
  - phys.derived.oxygenation_reserve  
  - phys.derived.exertion_index  

- phys.resp.etco2 →  
  - phys.derived.oxygenation_reserve  

- phys.temp.core →  
  - phys.derived.thermal_stress  

- phys.temp.skin →  
  - phys.derived.thermal_stress  

- phys.motion.imu →  
  - phys.derived.gait_stability  
  - phys.derived.exertion_index  

- phys.motion.emg →  
  - phys.derived.exertion_index  

- phys.neuro.eeg_bands →  
  - phys.derived.cognitive_load  
  - phys.derived.fatigue_index  

- phys.hydration.index →  
  - phys.derived.hydration_status  

- phys.hydration.na, phys.hydration.k, phys.hydration.sweat_rate →  
  - phys.derived.hydration_status  

- phys.metabolic.glucose, phys.metabolic.lactate, phys.metabolic.hb →  
  - phys.derived.metabolic_rate  

Derived metrics → composite states

- phys.derived.hrv, phys.derived.shock_index →  
  - state.cv  

- phys.derived.oxygenation_reserve →  
  - state.resp  

- phys.derived.thermal_stress →  
  - state.thermal  

- phys.derived.hydration_status →  
  - state.hydration  

- phys.derived.cognitiveload, phys.derived.fatigueindex, phys.derived.exertion_index →  
  - state.fatigue  

- state.cv, state.resp, state.thermal, state.hydration, state.fatigue, phys.derived.metabolic_rate →  
  - state.overall_health  

Composite states → control loops

- state.resp, phys.derived.oxygenation_reserve →  
  - ctrl.env.o2  
  - ctrl.env.co2_scrubber  

- state.thermal, phys.derived.thermal_stress →  
  - ctrl.env.thermal  

- state.hydration, phys.derived.hydration_status →  
  - ctrl.ops.hydration  

- state.fatigue, phys.derived.cognitive_load →  
  - ctrl.ops.workload  

- state.overall_health, all state nodes →  
  - ctrl.ops.alerts
